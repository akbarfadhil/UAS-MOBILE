Aplikasi Hadist Digital adalah aplikasi mobile yang dikembangkan menggunakan Flutter. Tujuannya adalah untuk memudahkan pengguna mencari dan membaca koleksi hadis dari berbagai sumber, serta menyediakan fitur pencarian cepat dan penandaan favorit.

Aplikasi ini berfokus pada:

Menampilkan koleksi hadis berdasarkan kategori/perawi.

Menyediakan fungsi pencarian teks lengkap dalam matan hadis.

Antarmuka pengguna yang bersih dan intuitif (UI/UX).
